mkdir -> criar pasta
rmdir -> excluir pasta
cd .. -> voltar diretório anterior
cd ~-> pasta raiz
cd diretório -> acessa uma pasta específica
touch nome.ext -> criar arquivo
rm -> deleta arquivo
nome_app -> executa o app
gnome-control-center -> abrir configurações
pwd -> mostra diretório atual
ls -> lista arquivos e diretórios
mv arquivo.ext /home/antonioirfilho/diretório -> move o arquivo
cp arquivo.ext /home/antonioirfilho/diretório -> copia arquivo e cola a copia em outro diretório especificado
